# 5x5德州扑克消除游戏设计方案

## 游戏核心概念

**5x5网格 + 德州扑克牌型 + 消除机制**

一个结合德州扑克的牌型判断和消除游戏机制的创意游戏。

---

## 设计方案一：经典消除模式

### 核心玩法
- 5x5网格，每个格子是一张扑克牌
- 玩家选择5张牌组成一手"牌"
- 组成有效德州扑克牌型即可消除得分

### 牌型判定
| 牌型 | 分数 |
|------|------|
| 同花顺 | 50分 |
| 四条 | 40分 |
| 葫芦 | 30分 |
| 同花 | 25分 |
| 顺子 | 20分 |
| 三条 | 15分 |
| 两对 | 10分 |
| 一对 | 5分 |

### 消除规则
- 选中的5张牌消除后，上方牌下落填补
- 顶部生成新牌
- 连锁消除有倍数加成

### 特色机制
- 每局有3次"换牌"机会（替换网格中任意一张牌）
- 特殊牌：万能牌（可当任意牌使用）

---

## 设计方案二：路径连接模式

### 核心玩法
- 5x5网格
- 玩家用手指/鼠标画出路径，连接相邻的牌（横竖斜均可）
- 路径上的牌组成牌型

### 创新点
- 不限定必须5张，3-7张都可以
- 牌型按最佳5张组合计算
- 路径越长，倍数越高

### 特殊规则
- 不能重复经过同一格
- 每回合有步数限制（如10步）
- 特殊格子：双倍格、锁定格、炸弹格

### 进阶机制
- 连击系统：连续做出牌型，倍数递增
- 挑战模式：限定时间内做出指定牌型

## 设计方案三：「All In」—— Roguelike 德州消除

**详细设计文档请查看：[ALL_IN_DESIGN.md](./ALL_IN_DESIGN.md)**

### 一句话介绍
**"在5x5的牌桌上，用德州扑克牌型击败赌场Boss，筹码构筑你的专属Build，体验一回合炸出顺子+同花+三条的破局爽感！"**

### 核心创新点
1. **交换消除**：像消消乐一样交换牌位置，行/列自动判定牌型
2. **筹码Build**：50+种被动道具，打造独特玩法
3. **连锁爽感**：一次操作触发多个牌型，伤害数字飞满屏幕
4. **Boss机制**：每个Boss都有独特技能，需要针对性策略

### 为什么这个模式好玩？
- **易上手**：5分钟学会交换牌位置
- **难精通**：50小时研究Build搭配和概率计算
- **每局不同**：Roguelike随机性，没有两局相同
- **极致爽快**：Build成型后一回合秒杀Boss的成就感

## 设计方案四：对战竞技模式

### 核心玩法
- 两人对战，共享5x5网格
- 轮流选择牌组成牌型
- 比大小决定胜负

### 对战规则
- 每轮双方同时选择5张牌
- 牌型大的玩家得分
- 使用的牌从网格中消除

### 策略深度
- 抢牌机制：双方都想用同一张牌时，速度快者得
- 阻挡策略：故意组成小牌型消耗好牌
- 读牌：预判对手想要什么牌

### 模式变体
- 锦标赛模式：多人淘汰制
- 合作模式：两人合作达成目标分数

---

## 设计方案五：解谜剧情模式

### 核心玩法
- 关卡制解谜游戏
- 每个关卡有特定目标（如：做出3个同花顺）
- 5x5网格初始固定，玩家需要思考最优解

### 关卡设计
- 教学关：基础牌型
- 挑战关：限定步数/时间
- 创意关：特殊规则（如：只能选黑色牌）

### 剧情包装
- 德州扑克主题的故事
- 角色收集与养成
- 不同角色有不同技能（如：可以透视一张牌）

## 推荐组合方案

基于《小丑牌》的成功经验，建议采用：

### 核心模式：方案三（Roguelike构筑）
- **深度最高**：不只是消消乐，还要考虑牌组构筑、资源管理、路线规划
- **重玩价值大**：每局都是随机生成，不会重复
- **符合独立游戏玩家喜好**：策略性强，有挑战性

### 附加模式：方案一（经典消除）
- **作为休闲模式**：不想动脑时可以玩
- **适合碎片时间**：随时开始，随时结束
- **新手教学**：帮助玩家学习基础牌型

### 关键差异化（为什么玩家会选择我们的游戏？）

#### 1. 独特的视觉体验
- **霓虹/复古风格**：类似《小丑牌》的美术风格
- **牌型动画**：做出同花顺时，5张牌会发光、旋转、炸裂
- **特效反馈**：每个牌型都有独特的视觉特效

#### 2. 丰富的音效设计
- **每张牌都有声音**：抽牌、选牌、消除都有不同的音效
- **牌型语音**：做出大牌型时，有激动人心的配音（"Royal Flush!!!"）
- **背景音乐**：紧张刺激的战斗音乐 + 轻松愉快的商店音乐

#### 3. 真正的策略深度
- **不只是找牌型**：要考虑牌堆还剩什么、敌人下回合会做什么、要不要留着好牌
- **构筑的乐趣**：像搭积木一样，慢慢打造属于你的最强牌组
- **随机性与技巧的平衡**：运气不好也能靠技术赢

#### 4. 社交与竞争
- **每日挑战**：每天一个固定种子，全球玩家比拼分数
- **排行榜**：看你在好友中排第几
- **分享功能**：做出神级牌型可以一键分享截图
