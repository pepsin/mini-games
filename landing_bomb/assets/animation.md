## 资源详细规格

### 1. 炸弹人 (bomb/)

**类型**: 动画 (`animation`)

**用途**: 从天空降落的敌人

#### 尺寸规格
| 属性 | 值 | 说明 |
|-----|---|------|
| 画布尺寸 | 64×64 px | 正方形画布 |
| 炸弹主体 | ~40×40 px | 圆形炸弹主体 |
| 安全边距 | 12 px | 四周留白 |

#### 动画帧要求
- **frame_01.png** - 正常状态，引信火花朝上
- **frame_02.png** - 轻微摆动，引信火花偏左
- **frame_03.png** - 正常状态，引信火花朝下
- **frame_04.png** - 轻微摆动，引信火花偏右

**播放速度**: 8 FPS (每帧 125ms)

**视觉元素**:
- 黑色圆形炸弹主体
- 白色骷髅脸（眼睛 + 嘴巴）
- 顶部引信带火花动画
- 暗灰色渐变增加立体感

---

### 2. 降落伞 (bomb/parachute/)

**类型**: 静态图 (`static`)

**用途**: 炸弹人的降落伞，连接在炸弹上方

#### 尺寸规格
| 属性 | 值 | 说明 |
|-----|---|------|
| 画布尺寸 | 96×96 px | 正方形画布 |
| 伞盖直径 | ~80 px | 半圆形伞盖 |
| 锚点位置 | 中心偏下 (0.5, 0.8) | 连接炸弹的位置 |

**视觉元素**:
- 红色半圆形伞盖
- 白色斜条纹装饰
- 4根灰色伞绳向下延伸
- 半透明效果

---

### 3. 花朵 (flower/)

**类型**: 动画 (`animation`) + 状态机

**用途**: 生命值显示，4朵花代表4条命

#### 尺寸规格
| 属性 | 值 | 说明 |
|-----|---|------|
| 画布尺寸 | 48×64 px | 高度包含茎部 |
| 花朵直径 | ~36 px | 花瓣外径 |
| 茎部长度 | ~28 px | 从花朵到底部 |
| 锚点位置 | (0.5, 0.8) | 茎部底部 |

#### 动画帧要求 - 存活状态 (alive)
**播放速度**: 4 FPS

- **alive_01.png** - 正常微笑，叶子自然状态
- **alive_02.png** - 微笑，叶子轻微摆动
- **alive_03.png** - 大笑（眼睛弯起），叶子展开
- **alive_04.png** - 回到微笑，叶子摆动（可用 alive_02.png）

#### 静态帧 - 死亡状态 (dead)
- **dead.png** - 枯萎花朵，灰色色调，悲伤表情

**颜色变体** (通过 tint 实现):
- 红色 `#FF5252` - 第1朵花
- 粉色 `#FF4081` - 第2朵花
- 橙色 `#FFAB40` - 第3朵花
- 紫色 `#E040FB` - 第4朵花

**视觉元素**:
- 6片花瓣围绕中心
- 金色花心带笑脸
- 绿色茎部 + 两片叶子
- 死亡状态：灰色调，花瓣下垂，悲伤表情

### 4. 云朵 (cloud/)

**类型**: 静态图 (`static`)，多版本

**用途**: 背景装饰，随机飘动

#### 尺寸规格
| 属性 | 小云朵 | 中云朵 | 大云朵 |
|-----|-------|-------|-------|
| 画布尺寸 | 128×64 px | 128×64 px | 128×64 px |
| 实际宽度 | ~60 px | ~90 px | ~120 px |
| 实际高度 | ~25 px | ~35 px | ~45 px |

**视觉元素**:
- 白色/浅灰色不规则形状
- 由3-4个椭圆组合而成
- 半透明 (alpha: 0.9)
- 柔和边缘

---

### 5. 彩虹 (rainbow/)

**类型**: 静态图 (`static`)

**用途**: 背景装饰

#### 尺寸规格
| 属性 | 值 | 说明 |
|-----|---|------|
| 画布尺寸 | 256×128 px | 半圆弧形 |
| 彩虹直径 | ~240 px | 完整半圆 |
| 锚点位置 | (0.5, 1.0) | 底部中心 |

**视觉元素**:
- 7色彩带（红橙黄绿蓝靛紫）
- 每条色带宽约 8-10 px
- 半透明度 (~40%)
- 柔和渐变边缘

### 6. 弹弓 (slingshot/)

**类型**: 静态图 (`static`)

**用途**: 玩家控制的射击装置

#### 尺寸规格
| 属性 | 值 | 说明 |
|-----|---|------|
| 画布尺寸 | 64×96 px | 包含完整弹弓 |
| 叉尖间距 | 36 px | 左右叉尖距离 |
| 叉尖高度 | 50 px | 从底部到叉尖 |
| 锚点位置 | (0.5, 1.0) | 底部中心 |

**关键坐标** (相对于弹弓中心):
- 左叉尖: `(-18, -50)`
- 右叉尖: `(18, -50)`
- 底部中心: `(0, 0)`

**视觉元素**:
- 棕色木质叉身
- 深棕色皮筋（在游戏代码中绘制）
- 圆形叉尖连接点

## info.json 格式规范

### 动画类型 (`"type": "animation"`)

```json
{
  "name": "资源名称",
  "type": "animation",
  "description": "描述",
  "fps": 8,
  "loop": true,
  "frames": [
    { "file": "frame_01.png", "duration": 125 },
    { "file": "frame_02.png", "duration": 125 }
  ],
  "size": { "width": 64, "height": 64 },
  "anchor": { "x": 0.5, "y": 0.5 }
}
```

### 带状态的动画 (`states`)

```json
{
  "name": "flower",
  "type": "animation",
  "states": {
    "alive": { "frames": [...] },
    "dead": { "frames": [...] }
  }
}
```

### 静态图 (`"type": "static"`)

```json
{
  "name": "资源名称",
  "type": "static",
  "description": "描述",
  "file": "image.png",
  "size": { "width": 96, "height": 96 },
  "anchor": { "x": 0.5, "y": 0.8 }
}
```

### 多版本静态图 (`variants`)

```json
{
  "name": "cloud",
  "type": "static",
  "variants": [
    { "id": "small", "file": "cloud_small.png" },
    { "id": "large", "file": "cloud_large.png" }
  ]
}
```

## 锚点说明

锚点决定图片绘制时如何定位到游戏坐标：

- `(0.5, 0.5)` - 中心点，图片中心对准游戏坐标
- `(0.5, 1.0)` - 底部中心，图片底部中心对准游戏坐标
- `(0.5, 0.8)` - 偏下位置，适合有茎/绳子的物体

计算公式：
```
drawX = gameX - width * anchor.x
drawY = gameY - height * anchor.y
```

## 导出建议

1. **格式**: PNG (支持透明)
2. **色彩模式**: RGBA
3. **推荐工具**: 
   - Aseprite (像素动画)
   - Photoshop
   - Figma
4. **命名规范**: 小写 + 下划线，如 `alive_01.png`
